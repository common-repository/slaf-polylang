Plugin allows restrict users to edit content on wp-admin only with specified selected language.

== Description ==

Plugin is extension for Polylang plugin. 

Plugin allows you to select which language only could be edited by specified users. 

Plugin creates extra field on user create and edit form, with all available languages from Polylang plugin. 
You can select, which language after login of specified user will be available to edit content and details.

In simply words, when you want restrict user to edit only one languge, plugin gives you this ablity.


Features:

*   Ability to limit used languages by user to one 
*   Ability to select all languges for user.
*   It works only for Admin Panel, so your webpage will be still available in all languages when you logged, and has language restriction.

== Installation ==

1. Upload plugin `Polylang - Specified language access` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create new user, or go to user edit section and setup languge restriction.

== Frequently Asked Questions ==

= I have 3 languages on Polylang and I want restrict user only to one. Could I do that? =

Not for now, current version gives you ability only to setup one language per one user. 
Mentioned feature will be available on next relase.
